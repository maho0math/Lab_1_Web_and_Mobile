# Lab_1_Web_and_Mobile
### Описание

В рамках данной работы мы реализовали минималистичный веб-сервер, который предоставляет API для получения количества дней до наступления Нового года. Мы изучили основы HTTP-протокола, принципы маршрутизации и процесс контейнеризации приложения.

### Технологический стек

* **Язык программирования:** Python 3.11

* **Фреймворк:** FastAPI

* **ASGI-сервер:** Uvicorn

* **Инструменты тестирования:** Postman

* **Контейнеризация:** Docker

### Ход выполненной работы

#### 1. Разработка сервера

Мы создали проект на базе FastAPI и настроили обработчик HTTP GET-запросов по маршруту /info.

#### 2. Реализация бизнес-логики

Мы написали алгоритм, который определяет текущую дату и вычисляет количество полных дней до 1 января следующего года. Для работы с датами была использована стандартная библиотека datetime.

#### 3. Контейнеризация

Мы подготовили Dockerfile для создания образа приложения, что позволяет запускать проект на любых устройствах без необходимости установки Python и зависимостей во внешнюю среду.

### Список команд для запуска

Ниже приведен полный список команд, которые мы использовали для подготовки и запуска проекта:

| № | Действие | Команда / Инструкция |

| :--- | :--- | :--- |

| **1** | **Подготовка** | Запустить **Docker Desktop** на компьютере. |

| **2** | **Сборка образа** | docker build -t new-year-timer . |

| **3** | **Запуск контейнера** | docker run -d -p 4200:4200 --name timer-app new-year-timer |

| **4** | **Тестирование** | Выполнить GET-запрос в **Postman** на http://localhost:4200/info |

### Пример взаимодействия с API

**Запрос:**

GET http://localhost:4200/info

**Пример ответа (JSON):**

{

&nbsp;   "days\_before\_new\_year": 302
